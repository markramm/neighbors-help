# About Neighbors Help

A neighborhood resource finder. Find food, shelter, medical care, and community resources near you.

## What this is

This site is a public directory of organizations that help neighbors. Pantries, free clinics, shelters, libraries, tool shares, crisis lines. Each resource has an address, hours, and a phone number. The map is color-coded by resource type so you can find what you need quickly.

## What this is not

- Not a social network. There are no accounts, no profiles, no messaging.
- Not a database of people. Only of organizations and the resources they offer.
- Not a tracker. There are no cookies, no fingerprinting, no behavioral analytics.

## Where the data comes from

We seed the directory from public sources — USDA SNAP, Feeding America, OpenStreetMap, and others. Community members submit corrections and additions through GitHub. A small maintainer team reviews every change.

The data is open. Anyone can [browse](https://github.com/markramm/neighbors-help/tree/main/kb), fork, or mirror it.
